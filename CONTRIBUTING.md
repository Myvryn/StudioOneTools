# Contributing to StudioOneTools

Thank you for your interest in contributing to **StudioOneTools**!  
This project welcomes community involvement, feature ideas, bug reports, and pull requests.

Please read the guidelines below to help ensure a smooth contribution process.

---

## 📢 Before You Begin
- Review the README for project context.
- Check existing **Issues** and **Pull Requests** to avoid duplicates.
- For major changes, please open an Issue first so we can discuss your idea.

---

## 🐛 Bug Reports
When reporting bugs, include the following:

- A clear description of the issue  
- Steps required to reproduce it  
- Expected vs. actual behavior  
- Operating system and Studio One version  
- Logs or screenshots (if applicable)

Please label the Issue with: `bug`.

---

## 💡 Feature Requests
To propose a new feature:

1. Search existing Issues to avoid duplicates.  
2. Open a new Issue labeled `enhancement`.  
3. Include:
   - What problem the feature solves  
   - How it improves workflow  
   - Any UI or UX suggestions  
   - Example use cases  

---
